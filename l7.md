### nearest neighbours for classification
to determine the class of a new example E:
- calculate the dist between E and all examples in the training set
- select the k nearest neighbours (smallest distances)
- determine the class of E by majority vote among the k neighbours

#### k-nearest neighbour algorithms
a.k.a instance-based learning (IBL) or lazy learning, memory-based learning
- no model is built: store all training examples
- any processing is delayed until a new instance must be classified -> lazy learning

##### example: k=3
