# data summarization

## measures of central tendency
- mean, mode, median, etc
    - cases for both 
        - median more robust but doesnt incorporate everyone
        - mean sensitive to outliers but includes everyone as part of the measure, etc
        - mode -- most common data-point, is good for categorical data

```python
df['column_name'].mean()
df['column_name'].median()
df['column_name'].mode()
# similarly:
df['column_name'].value_counts()

# prints a table of multiple summary statistics
df['column_name'].describe()
# or a full dataframe:
df.describe()

# df size:
df.shape

```

## Missing values

number of methods to deal with missing values in a data frame:
| df.method()   | description    |
|--------------- | --------------- |
| dropna()   | drop missing values   |
| dropna(how='all')   | drop values where all cells are NA   |
| dropna(axis=1,how='all')   | drop cl if all values are missing   |
| dropna(thresh=5)   | drop rows that contain less than 5 values   |
| dropna(inplace=True)   | drops na's and reassigns the dataframe to itself (good for modifying single cols)  |
| fillna(0)   | fill na entries with 0 |
|  isna()  | false when there is a value, true otherwise (when NA) |
